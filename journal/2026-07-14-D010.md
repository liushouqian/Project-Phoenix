## Sprint Task 8 - Session History Debug API

## Today's Goal
Add a history inspection API for Phoenix conversation memory.

The goal is to make conversation state ovservable and easier to debug.

## Background

Before this task, Phoenix already had:

- Chat API
- Conversation memory storage
- Session reset API

However, developers could not directly check  the current session history.

This task adds a way to inspect stored conversation history.

## Completed

- Added HistoryRequest schema
- Added HistoryResponse schema
- Added `/chat/history` API endpoint
- Connected history API with conversation_store
- Tested history reading after multiple conversations
- Tested history after session reset
